# 模型老K 使用指南

## 📖 简介

模型老K是一个基于T7概念资金双驱模型v2.0的智能选股推荐系统，采用概念竞速引擎 + 资金流验证 + 动态龙头筛选 + 筹码获利盘分析。系统通过多维度技术指标、市场状态识别、概念竞速分析、资金流验证和AI智能过滤，为投资者提供高质量的股票推荐。

**T7 v2.0 核心特性：**
1. **简化配置**：参数从20+个收敛到7个核心参数，操作更简单
2. **自动优化**：市场状态自动识别并调整策略，权重自动调整
3. **概念竞速引擎**：从单一行业进化为"最强概念竞速"，精准捕捉个股的真实上涨驱动力
4. **资金流因子**：引入主力资金净流入、超大单净流入等资金流数据验证主力意图
5. **动态龙头筛选**：基于最强概念的领涨股豁免机制
6. **筹码获利盘**：ASR和真空区判断
7. **多日概念热度**：考虑概念3日累计表现，提升推荐质量

## 🎯 核心功能

### 1. 今日推荐（智能推荐）

基于当前参数设置，获取符合条件的推荐股票。

**功能特点：**
- 支持选择历史日期进行推荐（留空则使用最近交易日）
- 自动处理非交易日，转换为最近交易日
- 显示推荐股票的详细信息：代码、名称、现价、AI打分、胜率、市场状态、板块、共振分、核心理由、止损价
- 提供诊断信息，帮助理解为什么没有找到符合条件的股票

**推荐结果字段说明：**
- **肥羊代码/名称**：股票代码和名称
- **现价**：推荐时的股票价格（买入价）
- **AI打分**：T7模型综合评分（0-100分）
  - 绿色：>50分，优质推荐
  - 橙色：30-50分，中等推荐
  - 红色：<30分，一般推荐
- **胜率**：AI预测的盈利概率（%）
- **市场状态**：当前市场状态（进攻/防守/震荡）
- **板块**：所属热门板块
- **共振分**：板块共振评分（正数表示共振，负数表示背离）
- **核心理由**：推荐的核心逻辑标签（如：倍量2.5倍 + 机器人板块龙头）
- **止损价**：建议的止损价格

**使用步骤：**
1. 在左侧"策略参数配置"面板调整策略参数
2. （可选）选择推荐日期，留空则使用最近交易日
3. 点击"智能推荐 (Get Alpha)"按钮
4. 查看推荐结果列表
5. 如果未找到符合条件的股票，查看诊断信息并调整参数

**诊断信息说明：**
- **Level 1**：数据库缺少足够的历史数据（至少需要90天）
- **Level 2**：MA条件过严或上市天数限制，建议放宽MA条件（如从MA60改为MA20）或关闭MA限制
- **Level 4**：AI过滤过严或胜率要求过高，建议关闭AI过滤或降低胜率要求
- **市场状态**：当前市场状态可能不适合当前策略，建议根据市场状态调整参数
- **板块共振**：当前可能缺乏板块共振，建议关注主线板块

### 2. 回测仪表盘（Time Travel）

通过历史数据回测，验证策略在不同市场环境下的表现。

**功能特点：**
- 支持自定义回测时间范围（最多6个月）
- 自动计算策略关键指标：胜率、爆款率、总收益率、最大回撤
- 可视化资金曲线，直观展示策略收益变化
- 帮助优化参数设置，找到最佳参数组合

**使用步骤：**
1. 在左侧"策略参数配置"面板设置策略参数
2. 选择回测时间范围（开始日期和结束日期）
3. 点击"执行回测 (Time Travel)"按钮
4. 查看回测结果：
   - **胜率**：盈利交易占比
   - **爆款率**：5日涨幅>10%的交易占比
   - **总收益率**：策略累计收益率
   - **最大回撤**：策略最大亏损幅度
   - **资金曲线**：策略收益随时间的变化趋势

### 3. 历史战绩

查看所有历史推荐记录及其验证结果。

**功能特点：**
- 自动保存每次推荐记录
- 自动验证早期推荐（推荐日期距离今天超过5个交易日）
- 显示推荐时的参数快照
- 显示验证结果：后5日最大涨幅、后5日涨幅、成功/失败标记
- 支持清空失败记录或清空所有记录

**历史记录字段说明：**
- **推荐日期**：推荐计算的日期
- **肥羊**：股票代码和名称
- **当时参数**：推荐时使用的参数快照（倍量、RPS等）
- **后5日最大涨幅**：推荐后5个交易日内达到的最大涨幅
- **后5日涨幅**：推荐后5个交易日的最终涨幅
  - 红色标签：涨幅为正
  - 绿色标签：涨幅为负
  - 灰色标签：未验证
- **结果**：验证结果
  - 绿色"成功"：后5日涨幅>5%
  - 红色"失败"：后5日涨幅≤5%
  - 灰色"未验证"：推荐日期距离今天不足5个交易日

**使用步骤：**
1. 切换到"历史战绩"标签页
2. 查看历史推荐记录列表
3. 分析成功和失败的案例，优化参数设置
4. （可选）点击"清空失败记录"或"清空所有记录"清理历史数据

## ⚙️ 参数配置说明

**v2.0 简化说明**：T7 v2.0已将参数从20+个收敛到7个核心参数，其他参数由系统自动处理。用户只需关注核心参数即可，系统会根据市场状态自动调整策略。

### 基础筛选（3个参数）

#### 市值范围（亿元）

**说明**：筛选股票市值范围，排除过大或过小的股票。

**默认值**：50-300亿

**建议设置：**
- 小盘股：10-100亿（波动大，风险高）
- 中盘股：50-300亿（平衡风险和收益，推荐）
- 大盘股：200-1000亿（波动小，稳健）

#### RPS阈值

**说明**：相对强度排名阈值，筛选出相对市场表现更强的股票。

**范围**：50-100

**默认值**：80（v2.0已优化，从90降低到80）

**建议设置：**
- 50-75：筛选范围较宽，推荐数量多
- 75-85：平衡选择（推荐）
- 85-100：筛选范围很窄，只推荐最强股票

#### 倍量定义

**说明**：定义"倍量"的倍数，当日成交量需要达到均量的多少倍才算倍量。

**范围**：1.0x - 5.0x

**默认值**：1.5x（v2.0已优化，从2.0x降低到1.5x）

**建议设置：**
- 1.0x-1.5x：较宽松，推荐数量多
- 1.5x-2.5x：平衡选择（推荐）
- 2.5x-5.0x：很严格，只推荐放量明显的股票

### 核心参数（2个参数）

#### 涨幅范围

**说明**：震荡模式下的涨幅范围（最小涨幅 ~ 最大涨幅）。系统会根据市场状态自动调整：
- **进攻模式**：自动放宽涨幅范围，允许追高
- **防守模式**：自动限制涨幅上限，避免追高
- **震荡模式**：使用用户设置的涨幅范围

**默认值**：2.0% ~ 9.5%

**建议设置：**
- 最小涨幅：0%-3%（较宽松）或 2%-5%（较严格）
- 最大涨幅：8%-10%（避免涨停）

**注意**：市场状态会自动识别并调整涨幅范围，用户只需设置震荡模式的基准值即可。

#### 概念共振

**说明**：是否启用概念共振加分。启用后，系统会优先推荐有概念共振的股票，提升推荐质量。

**默认值**：开启

**建议**：
- 开启：推荐质量更高，关注概念热点（推荐）
- 关闭：只使用技术指标筛选，推荐数量可能更多

### AI过滤（2个参数）

#### AI过滤

**说明**：是否启用AI智能过滤，使用AI模型进一步筛选推荐股票。

**默认值**：开启

**建议**：
- 开启：AI会过滤掉一些技术面看似符合但基本面或市场情绪不佳的股票，推荐质量更高（推荐）
- 关闭：只使用技术指标筛选，推荐数量可能更多，但质量可能参差不齐

#### 最小胜率

**说明**：AI预测的胜率阈值，低于此值的股票将被过滤。

**范围**：40%-70%

**默认值**：45%（v2.0已优化，从50%降低到45%）

**建议设置：**
- 40%-45%：较宽松，推荐数量多（推荐）
- 45%-55%：平衡选择
- 55%-70%：很严格，只推荐高胜率股票

### 自动处理参数（无需配置）

以下参数由系统自动处理，用户无需配置：

1. **市场状态识别**：系统自动识别市场状态（进攻/防守/震荡），并自动调整策略
2. **权重调整**：根据市场状态自动调整技术/趋势/热度权重
3. **涨幅范围调整**：根据市场状态自动调整涨幅范围
4. **通用过滤**：自动剔除新股（上市<60天）、ST股票、涨停股票
5. **量比/上影线/VWAP等**：已内置优化，无需单独配置

**提示**：v2.0版本大幅简化了参数配置，用户只需关注上述7个核心参数即可，系统会自动处理其他复杂逻辑。

### 推荐日期

**说明**：选择推荐计算的日期，留空则使用最近的交易日。

**使用场景：**
- 留空：获取今日推荐（使用最近交易日的数据）
- 选择历史日期：回测历史推荐效果，验证策略在特定日期的表现

**注意**：不能选择未来的日期

## ⚠️ 注意事项

1. **数据依赖**
   - 系统需要足够的历史数据（至少90天）
   - 确保数据库中有最新的股票数据
   - 非交易日会自动转换为最近交易日

2. **推荐时效性**
   - 推荐结果基于推荐日期的数据
   - 市场情况变化后，推荐结果可能不再适用
   - 建议及时关注市场变化

3. **风险提示**
   - 本系统仅提供技术面分析，不构成投资建议
   - 投资有风险，决策需谨慎
   - 建议结合基本面分析、市场环境等因素综合判断
   - 设置止损价，控制风险

4. **参数调整**
   - 参数调整会影响推荐结果
   - 建议先回测验证参数效果
   - 不要频繁大幅调整参数

5. **历史战绩**
   - 历史战绩仅供参考，不代表未来表现
   - 市场环境变化会影响策略效果
   - 需要持续优化和调整

6. **市场状态**
   - 系统会自动识别市场状态（进攻/防守/震荡）
   - 不同市场状态使用不同的筛选策略
   - 建议根据市场状态调整参数

## 🔧 常见问题

### Q1: 为什么没有找到符合条件的推荐股票？

**可能原因：**
1. 参数设置过严（如RPS阈值过高、倍量要求过高、涨幅范围过窄）
2. 市场环境不佳，符合条件的股票较少
3. 数据库缺少足够的历史数据（至少需要60天）
4. 市场状态不适合当前策略

**解决方法：**
1. 查看诊断信息，了解具体原因
2. 适当放宽参数（降低RPS阈值、降低倍量要求、放宽涨幅范围）
3. 尝试关闭AI过滤或降低胜率要求
4. 系统会自动根据市场状态调整策略，无需手动切换
5. 选择其他日期进行推荐

### Q2: 回测结果很好，但实际推荐效果不佳？

**可能原因：**
1. 市场环境发生变化
2. 参数过度优化（过拟合）
3. 样本量不足
4. 市场状态发生变化

**解决方法：**
1. 使用更长时间范围回测
2. 避免过度优化参数
3. 结合当前市场环境调整参数
4. 关注市场状态变化
5. 持续跟踪历史战绩，及时调整

### Q3: 如何判断推荐股票的质量？

**参考指标：**
1. **AI打分**：>50分为优质推荐
2. **胜率**：>60%为较高胜率
3. **市场状态**：与当前市场状态匹配
4. **板块共振**：共振分为正数，表示有板块共振
5. **核心理由**：包含多个正面标签（倍量+板块+技术形态）
6. **历史战绩**：查看类似参数的历史表现

### Q4: 推荐日期选择有什么作用？

**作用：**
1. 回测历史推荐效果
2. 验证策略在特定日期的表现
3. 分析不同市场环境下的策略效果

**使用建议：**
- 选择重要市场节点（如大跌后、大涨后）
- 选择不同市场环境（进攻、防守、震荡）
- 对比不同日期的推荐结果

### Q5: 如何清空历史记录？

**方法：**
1. 切换到"历史战绩"标签页
2. 点击"清空失败记录"：只删除失败的推荐记录
3. 点击"清空所有记录"：删除所有历史记录

**注意**：清空操作不可恢复，请谨慎操作。

### Q6: 市场状态如何识别？需要手动调整吗？

**说明：**
- 系统使用RSRS指标**自动识别**市场状态
- 市场状态分为：进攻（Attack）、防守（Defense）、震荡（Balance）
- 不同市场状态**自动使用**不同的筛选策略
- 推荐结果中会显示当前市场状态

**v2.0优化：**
- **无需手动调整**：系统会自动识别市场状态并调整策略
- **自动调整涨幅范围**：进攻模式允许追高，防守模式限制涨幅
- **自动调整权重**：进攻模式重技术，防守模式重概念
- **自动处理龙头豁免**：防守模式下概念龙头自动豁免涨幅限制

**建议：**
- 只需关注推荐结果中的市场状态标签
- 在进攻市场时，系统会自动放宽涨幅，关注概念共振
- 在防守市场时，系统会自动限制涨幅，控制风险
