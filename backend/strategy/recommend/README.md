# 推荐策略模块 (Recommendation Strategy)

本目录包含智能选股与策略回测的插件实现。

## 架构说明

采用插件化架构，由 `engine.py` 统一调度。所有策略插件均继承自 `plugins/base.py`。

## 策略插件

### 1. Alpha 动量策略 (`plugins/alpha_momentum.py`)

**核心逻辑**：结合价格动量 (RPS) 与业绩稳健性 (Sharpe Ratio) 的双因子选股模型。

- **RPS (相对价格强度)**：要求年线 RPS > 90 或月线 RPS > 85。确保选出的股票是市场中的强者。
- **Sharpe Ratio (夏普比率)**：优先选择波动率低、上涨趋势稳定的个股，排除暴涨暴跌的妖股。
- **Bias (乖离率)**：过滤短期涨幅过大的标的 (Bias20 < 10%)，控制追高风险。
- **主线过滤**：强制要求标的必须属于当前市场的主线板块 (通过 `MainlineAnalyst` 判定)。

### 2. 回测验证插件 (`plugins/backtester.py`)

**功能**：
- **收益跟踪**：自动计算历史推荐标的在 T+1, T+3, T+5 的实际涨幅。
- **胜率统计**：验证"市场情绪指标"发出的 BUY/SELL 信号是否准确。

### 3. 科创50 ETF 回测 (`plugins/backtest_star50.py`)

**功能**：独立的向量化回测脚本。
- **对象**：科创50 ETF (000688.SH)。
- **信号**：基于 `market_sentiment` 表中的情绪标签 (如"共振进攻"、"恐慌错杀") 进行全仓/空仓轮动。
- **目的**：验证宏观情绪择时策略在指数ETF上的有效性。
